## Creating a plug

A plug is a collection of MCP servers you can create [here](https://mcplug.ai/dashboard).

First browse and find MCP servers that you want to use. Then add it to your plug.

You can manage your plugs on the [mcplug dashboard](https://mcplug.ai/dashboard).

## MCPlug Token

In order to use the mcplug client, you need to have a MCPlug token.

You can get a MCPlug token from the [mcplug dashboard](https://mcplug.ai/dashboard).
